## Clonar repositorio

```bash 
git clone https://github.com/YormanAlexanderCarrillo/ProyectoGradoBackend.git
```
## Ingresar carpeta

```bash
cd ProyectoGradoBackend
```

## Instalar paquetes necesarios

```bash
pip install
```

## Inicializar el servidor 
### Modo desarrollo

```bash
flask run --host=0.0.0.0 --port=5000
```
